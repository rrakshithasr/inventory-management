# 📦 Inventory Management System

A simple **Inventory Management System** built with **Python** and **Pandas**. This console-based application allows users to manage product inventory efficiently by adding, updating, searching, deleting, selling, and restocking products. All inventory data is stored in a CSV file, making it easy to save and retrieve information.

---

## 🚀 Features

* ➕ Add new products
* 📋 View all products
* 🔍 Search products by Product ID
* ✏️ Update product details
* ❌ Delete products
* 🛒 Sell products (automatically updates stock)
* 📦 Restock products
* ⚠️ Display low-stock products
* 💰 Calculate total inventory value
* 💾 Automatic CSV file storage
* 🛡️ Basic input validation and error handling

---

## ▶️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Inventory-Management-System.git
```

### 2. Open the project folder

```bash
cd Inventory-Management-System
```

### 3. Install the required library

```bash
pip install pandas
```

### 4. Run the program

```bash
python inventory.py
```
